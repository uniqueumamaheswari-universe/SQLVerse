# 🧠 AI Error / Hallucination Log

**Use this structure every time you catch the AI making a mistake – wrong syntax, missing edge case, logical contradiction, or hallucinated feature.**

Store each occurrence as a separate entry inside `Socratic_Journals/`. You can use one file with multiple headings, or separate files named `error1.md`, `hallucination_selfjoin.md`, etc.

---

## Hallucination Entry Template

```markdown
# Entry: [Date] – [Brief description]

**What the AI said:**  
(Quote the incorrect suggestion or logic)

**What was actually correct:**  
(Explain the right logic or SQLite syntax)

**How I caught it:**  
(What check, question, or edge case revealed the error)

**What I learned:**  
(One sentence)

**Source context (optional):**  
- Concept: [e.g., Self Join]  
- Database: [e.g., tourism_planet_self_join.db]
```
