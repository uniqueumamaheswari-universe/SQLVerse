


# Browser Office: Your First SQL Query

Welcome to your first day in the Browser Office! This file contains the validation query you will execute in **Tab 2: The Factory** (SQLite Online).

## Purpose
This simple query proves your `training_institution_sample.db` database is loaded correctly and your Factory environment is operational. It selects the first three records from the `students` table.

## The Query
Copy **only** the SQL command below from the code block.

```sql
SELECT * FROM students LIMIT 3;
```

> **⚠️ Critical Step:** When you switch to **Tab 1 (The Map)** to get this query, you must select and copy **only the line of SQL code above** (`SELECT * FROM students LIMIT 3;`). Do **not** copy the entire file or any markdown formatting. Paste only this line into the SQL editor in **Tab 2**.

## Next Steps
1.  After copying the query, switch to **Tab 2 (The Factory)**.
2.  Paste the query into the SQL editor.
3.  Click **Run** or press **F9**.
4.  You should see 3 rows of student data appear, confirming your setup is successful.

**Pro Tip:** This "find → copy → execute" workflow mirrors exactly how you will use the Browser Office for all future modules.
