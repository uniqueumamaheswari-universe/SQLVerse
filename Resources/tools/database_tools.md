# Database Tools

## 🗄️ SQLite (Levels 1-2)
- **DB Browser for SQLite** ([sqlitebrowser.org](https://sqlitebrowser.org/)) - Visual database management
- **SQLite Online Playgrounds**:
  - [SQLite Playground](https://sqlime.org/)
  - [SQL Fiddle](https://sqlfiddle.com/)

## ☁️ PostgreSQL (Level 3)
- **Neon.tech** ([neon.tech](https://neon.tech)) - Free serverless Postgres
- **Supabase** ([supabase.com](https://supabase.com)) - Free tier with extras
- **pgAdmin** - Official administration tool
- **DBeaver** - Universal database tool

## 🏢 SQL Server (Level 3)
- **Azure SQL Database** - Free tier for 12 months
- **Azure Data Studio** - Cross-platform tool
- **SQL Server Management Studio** (SSMS) - Windows management

## 🛠️ Database Utilities
- **dbdiagram.io** - Database schema design
- **CSV to SQL Converters** - For sample data import
